# Claude Code 가이드: 개요 및 빠른 시작

## Claude Code란?

Claude Code는 **Anthropic에서 공식으로 제공하는 에이전틱(agentic) 코딩 CLI 도구**입니다. 터미널에서 직접 실행되며, 여러분의 코드베이스를 이해하고, 파일을 편집하고, 명령어를 실행하며, 복잡한 코딩 작업을 자율적으로 수행합니다.

기존의 코드 자동완성 도구와 달리, Claude Code는 **대화형 에이전트**로 동작합니다. 단순히 코드 한 줄을 제안하는 것이 아니라, 프로젝트 전체를 파악하고 여러 파일에 걸친 변경 작업을 스스로 계획하고 실행할 수 있습니다.

### 일반적인 코드 어시스턴트와의 차이점

| 특성 | 일반 코드 어시스턴트 | Claude Code |
|------|---------------------|-------------|
| 동작 방식 | 코드 자동완성/제안 | 자율적 에이전트 |
| 작업 범위 | 단일 파일, 단일 함수 | 프로젝트 전체, 다중 파일 |
| 실행 환경 | IDE 플러그인 | 터미널 (CLI) |
| 명령 실행 | 불가 | 터미널 명령어 직접 실행 |
| Git 통합 | 제한적 | 커밋, PR, 코드 리뷰 등 완전 통합 |

## 주요 기능 요약

### 1. 에이전틱 코딩 (Agentic Coding)
Claude Code는 여러분이 자연어로 작업을 설명하면, 스스로 계획을 세우고 실행합니다.

```
> 이 프로젝트에 사용자 인증 기능을 추가해줘. JWT 토큰 방식으로.
```

위와 같이 요청하면 Claude Code는:
- 프로젝트 구조를 분석합니다
- 필요한 패키지를 설치합니다
- 인증 관련 파일들을 생성합니다
- 기존 라우터에 미들웨어를 추가합니다
- 테스트 코드까지 작성합니다

### 2. 다중 파일 편집 (Multi-file Editing)
여러 파일을 동시에 수정할 수 있습니다. 리팩토링, 기능 추가, 버그 수정 등 프로젝트 전반에 걸친 변경 작업을 한 번에 처리합니다.

### 3. Git 통합 (Git Integration)
커밋 메시지 작성, 브랜치 관리, Pull Request 생성, 코드 리뷰 등 Git 관련 작업을 자연어로 요청할 수 있습니다.

```
> 지금까지 변경한 내용을 커밋해줘. 커밋 메시지는 한국어로.
```

### 4. 터미널 접근 (Terminal Access)
Claude Code는 여러분의 터미널에서 직접 명령어를 실행할 수 있습니다. 빌드, 테스트, 패키지 설치 등 모든 터미널 작업을 수행합니다.

### 5. 코드베이스 이해 (Codebase Understanding)
프로젝트 전체를 분석하여 구조, 의존성, 패턴을 이해합니다. "이 프로젝트의 아키텍처를 설명해줘"와 같은 질문에 답할 수 있습니다.

## 빠른 시작 (Quick Start) - 3단계

### 1단계: 설치

```bash
npm install -g @anthropic-ai/claude-code
```

> **참고:** Node.js 18 이상이 필요합니다. 설치되어 있지 않다면 [Node.js 공식 사이트](https://nodejs.org)에서 먼저 설치하세요.

### 2단계: 인증

```bash
claude
```

처음 실행하면 인증 과정이 시작됩니다. Anthropic API 키를 입력하거나 OAuth를 통해 로그인할 수 있습니다.

### 3단계: 코딩 시작

프로젝트 디렉토리로 이동한 후 Claude Code를 실행합니다:

```bash
cd your-project
claude
```

이제 자연어로 작업을 요청할 수 있습니다:

```
> 이 프로젝트의 구조를 설명해줘
> README.md를 작성해줘
> 테스트를 실행하고 실패하는 테스트를 수정해줘
```

## 이 가이드의 대상

이 가이드는 다음과 같은 분들을 위해 작성되었습니다:

- **프로그래밍 입문자**: 코딩을 시작했지만 AI 도구의 도움을 받고 싶은 분
- **주니어 개발자**: 일상적인 개발 작업을 더 효율적으로 수행하고 싶은 분
- **시니어 개발자**: 반복적인 작업을 자동화하고 생산성을 극대화하고 싶은 분
- **팀 리더**: 팀에 Claude Code 도입을 검토하고 있는 분

> **사전 지식:** 기본적인 터미널 사용법과 Git에 대한 이해가 있으면 좋지만, 필수는 아닙니다. 이 가이드에서 필요한 내용을 함께 설명합니다.

## 주요 용어 빠른 참조

| 용어 | 설명 |
|------|------|
| **CLI (Command Line Interface)** | 터미널에서 텍스트 명령어로 조작하는 인터페이스. Claude Code는 CLI 도구입니다. |
| **에이전틱 (Agentic)** | AI가 스스로 계획을 세우고, 도구를 사용하며, 여러 단계의 작업을 자율적으로 수행하는 방식 |
| **프롬프트 (Prompt)** | Claude Code에 전달하는 자연어 명령이나 질문 |
| **CLAUDE.md** | 프로젝트 루트에 두는 설정 파일. Claude Code에게 프로젝트의 규칙과 맥락을 알려줍니다 |
| **컨텍스트 (Context)** | Claude Code가 현재 대화에서 기억하고 있는 정보의 범위 |
| **컨텍스트 윈도우 (Context Window)** | Claude가 한 번에 처리할 수 있는 텍스트의 최대 크기 |
| **토큰 (Token)** | AI 모델이 텍스트를 처리하는 기본 단위. 대략 한국어 한 글자가 1~2토큰입니다 |
| **MCP (Model Context Protocol)** | 외부 도구 및 서비스와 Claude Code를 연결하는 프로토콜 |
| **OAuth** | 비밀번호 대신 인증 토큰을 사용하는 인증 방식 |
| **API 키 (API Key)** | Anthropic API에 접근하기 위한 인증 문자열 |
| **슬래시 커맨드 (Slash Command)** | `/help`, `/clear` 등 슬래시(/)로 시작하는 Claude Code 내장 명령어 |
| **퍼미션 (Permission)** | Claude Code가 파일 수정이나 명령어 실행 시 필요한 권한 |
